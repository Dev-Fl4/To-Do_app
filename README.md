# To-DO_App

## how to use

### fork the repository
```
git clone "https://github.com/JuanB-D/To-Do_app.git
```
### Install dependeces 
```
npm install

### make a .env file with the next info
```
API_KEY=
MONGODB_URI=
JWT_SECRET_KEY=
PORT= //optional
```

# scripts

1. npm start // normally with node
2. npm run dev // starts with nodemon
